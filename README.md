## Review It - Chrome Extension

This is a simple Chrome (Manifest V3) extension written in TypeScript. It shows a popup with:

- **Title**: "Generate a review:"
- **5-star rating**: 1 star = bad experience, 5 stars = great experience.
- **Hover effect**: stars get a subtle highlight on hover.
- **Selection behavior**: when a star is selected, that star and all stars below it "shine".
- **Free text box**: placeholder "Generate a review ...".
- **Buttons**: blue `Cancel` and `Finish` buttons.

### Development

1. **Install dependencies**

```bash
npm install
```

2. **Build TypeScript**

```bash
npm run build
```

This will compile files from `src` into `dist`.

### Load the extension in Chrome

1. Run `npm run build`.
2. Open `chrome://extensions` in Chrome.
3. Enable **Developer mode** (top-right).
4. Click **Load unpacked** and choose this project folder.
5. The extension icon will appear in the toolbar. Click it to open the popup.

### Backend

Reviews are generated by your backend API (see `fetchReview` in `src/popup.ts`). No API key is stored in the extension.

### Upload to Chrome Web Store

```bash
zip -r review-it-store.zip manifest.json popup.html dist/popup.js icons/icon16.png icons/icon32.png icons/icon48.png icons/icon128.png
```

### Privacy policy (GitHub Pages)

The policy lives at [`docs/privacy.html`](docs/privacy.html). To publish it:

1. Push the repo to GitHub.
2. Open the repo → **Settings** → **Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Choose branch **main** (or **master**) and folder **`/docs`**, then **Save**.

Your public URL will be:

`https://<your-github-username>.github.io/<repo-name>/privacy.html`

Replace the placeholder contact email and the **Retention** paragraph in `docs/privacy.html` with your real details before publishing.
